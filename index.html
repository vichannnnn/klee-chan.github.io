Commands

Commands Explanations
=w=ghoststatus Checks if Ghost Ping Alert is on for the channel. By default, Ghost Ping Alert is on for all channels that Ghost-chan has access to.
If OFF, Ghost-chan will do nothing in the said channel.
=w=ghostoff Turns off Ghost Ping Alert for the chat channel the command was called in, when turned off, Ghost Ping Alert will never trigger in the said channel until it is turned on again.
=w=ghoston Turns on Ghost Ping Alert for the chat channel the command was called in if it was off, when turned on, Ghost Ping Alert will function normally in the said channel until it is turned off again.
=w=upload <Emoji Name> This command has to be used with an image attachment, it will upload the image that you've sent with the command as an emoji on your server, with the name being the emoji name you've entered. Manage Emojis Permission required.
=w=flip <Message ID> This command flips the image attached to the message ID horizontally.
=w=rotate <Message ID> This command rotates the image attached to the message ID by 90 degrees. There will be two directional reactions attached to the image and each time you react, it rotates 90 degrees clockwise/anticlockwise.
=w=ud <Words> Shows a list of urban dictionary definition of the said word, the list can be navigated through the reactions attached to the message.
=w=<action> <user:mention>
Available actions: hug / slap / kiss / pat / poke / bite / lick 		This command does whatever you choose to the user that you mentioned!
=w=google <Words> Shows the top 10 Google Search Result of the specified word.
=w=vote Vote for Ghost-chan at top.gg! Thank you for your support! :)
=w=colourenable Enables the Ghost-chan's Colour System! For more information on the colour system, use =w=help and navigate to the Colour category! Requires Administrator Permission.
=w=colourdisable Disables the Ghost-chan's Colour System! The colour system is by default disabled until you enable it on your server. Requires Administrator Permission.
=w=colour <0x hexcode> Example: =w=colour &0xff0000> Adds a customized colour role to yourself in the server. It will automatically replace if you already have a colour role from Ghost-chan. Use the remove argument to remove the colour completely.
=w=roleinfo <role mention> Shows the information of the specified role.
=w=userinfo <user mention (or not)> Shows the information of the specified user. Leave argument empty to show information about yourself.
=w=serverinfo Shows the information of the server.
=w=av <user mention (or not)> Shows the enlarged avatar of the specified user. Leave argument empty to show your avatar.
